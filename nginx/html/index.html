01. 개발환경 구축

Windows

PowerShell & WSL2 Ubuntu

Docker Desktop을 통해 WSL2 리눅스 커널 위에서 고성능 도커 환경을 구축합니다.
macOS

Apple Silicon & Intel

Virtualization.framework를 활용하여 매끄러운 개발 경험을 제공합니다.
Linux

AL2023 & Ubuntu 24.04

네이티브 환경에서 `apt` 또는 `dnf` 패키지 매니저로 엔진을 직접 설치합니다.

02. Docker CLI

컨테이너 생태계를 제어하는 핵심 명령어들을 익힙니다.

docker run

컨테이너 실행

docker network

가상 네트워크 관리

docker volume

영구 데이터 저장

03. Docker Build

나만의 애플리케이션 이미지를 설계합니다.

Dockerfile 내의 지시어(FROM, RUN, COPY, CMD)를 이해하고, 멀티 스테이지 빌드를 통해 이미지 용량을 최적화하는 방법을 배웁니다.

# Build command
docker build -t app:v1 .

04. Docker Compose


version: "3.9"
services:
  web:
    image: nginx:alpine
    ports:
      - "80:80"
  backend:
    build: ./api
    depends_on:
      - db

docker-compose up 한 번으로 복잡한 스택을 동시에 기동합니다.
